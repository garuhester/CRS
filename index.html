自定义随机选择器
CRS(Custem Random Selector)
使用方法：
1. 选择文件：你可以选择一个包含候选项的txt文件，选择器会自动添加文件中的候选项 ( 文件格式： 每一个候选项占一行即可，回车换行 ) 。
2. 添加按钮：在输入框中输入您想添加的候选项，点击添加按钮后即可将其加入到下方的列表中。
3. 删除按钮：根据下方列表，在输入框中输入候选项的序号，点击删除按钮后即可将其从列表中移除。
4. 生成按钮：点击生成按钮后，选择器会根据列表随机给您推荐一个候选项。
5. 清除按钮：点击清除按钮后，会将下方的列表以及本地储存删除。
6. 保存按钮：由于数据保存在本地浏览器中，当您每次修改数据或者需要在另外的浏览器和电脑上使用您的数据时，可以点击保存按钮将数据保存为txt文件，并下载下来，方便下次的使用。
